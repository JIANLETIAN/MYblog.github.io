c++笔记

发表于 2021-06-10 更新于 2021-02-25

c++初识

  • 基本格式

  • 变量定义和输出

  • 常量

    作用：用于记录程序中不可更改的数据

    1.#define宏常量

    #define 常量名 常量值
    通常在文件上方（主函数外）定义，表示一个常量

    2.const修饰的变量

    const 数据类型 常量名 = 常量值
    通常在变量定义前加关键字const,修饰变量为常量，不可修改

    在给变量或常量起名称时，不要用C++的关键字，否则会出错，且产生歧义

  • 标识符

    标识符不可以是关键字
    标识符由字母，数字，下划线构成
    标识符第一个字符只能是字母或下划线
    标识符是区分大小写的
    给变量起名时最好能做到见名知意

    数据类型

  • 整形

    作用：整形变量表示的是整数类型的数据

数据类型 占用空间 取值范围
short(短整型) 2字节 (-2^15~2^15-1)
int(整型) 4字节 (-2^31~2^31-1)
short(长整型) windows为4字节，linux为4字节(32位)，8字节(64位) (-2^31~2^31-1)
short(长长整型) 8字节 (-2^63~2^63-1)

  • 字符型

    char 字符名称='a'
    用单引号只能括一个

  • 转义字符

    \n 换行符
    \\ 显示
    \t 水平制表，跳到下一个Tab的位置
  • ##字符串类型
    要包含string头文件 #include <string>

  • 布尔类型

    ture 真 1
    false 假 0
    bool 布尔名称 = ture;

    cin >> 变量
